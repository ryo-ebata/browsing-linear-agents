Linear Agent Setup

Welcome to the Linear Agent Setup

This application helps you set up and configure a Linear Agent that can interact with your Linear workspace.

Linear Agents can:

  • Be mentioned in issues and comments
  • Be assigned to issues
  • Create and reply to comments
  • Move issues between states
  • And much more!
Install Linear Agent

Setup Instructions

  1. Create a Linear OAuth application in your Linear workspace
  2. Configure the application with the following settings:
    • Redirect URI: http://localhost:3000/auth/callback
    • Enable webhooks and select "Inbox notifications"
  3. Copy your Client ID and Client Secret to your .env file
  4. Start the server with npm start
  5. Click the "Install Linear Agent" button above

Environment Configuration

Make sure to set up your .env file with the following variables:

LINEAR_CLIENT_ID=your_client_id
LINEAR_CLIENT_SECRET=your_client_secret
LINEAR_REDIRECT_URI=http://localhost:3000/auth/callback
LINEAR_WEBHOOK_SECRET=your_webhook_secret
PORT=3000
NODE_ENV=development
