Linear Agent Setup

Welcome to the Linear Agent Setup

This application helps you set up a Linear Agent that can interact with your Linear workspace.

Linear Agents can be mentioned, assigned to issues, and respond to various events in your workspace.

Install Linear Agent

Setup Instructions

  1. Click the "Install Linear Agent" button above
  2. Authorize the application in Linear
  3. After successful installation, you'll see your App User ID
  4. Add this ID to your .env file as LINEAR_APP_USER_ID
  5. Restart the server

Using Your Agent

Once installed, you can:

  • Mention the agent in issues and comments using @YourAgentName
  • Assign issues to the agent
  • The agent will respond based on the configured handlers

Configuration

Make sure your .env file contains the following:

LINEAR_CLIENT_ID=your_client_id
LINEAR_CLIENT_SECRET=your_client_secret
LINEAR_REDIRECT_URI=http://localhost:3000/auth/callback
LINEAR_WEBHOOK_SECRET=your_webhook_secret
LINEAR_APP_USER_ID=app_user_id_after_installation
LINEAR_TOKEN=your_access_token
